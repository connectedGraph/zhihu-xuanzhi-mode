# 📜 知乎宣纸沉浸阅读脚本 (README)

## 📖 简介

“一叶孤舟，一卷宣纸”。本脚本专为知乎深度阅读爱好者打造。一键将知乎繁杂的现代网页界面转化为极简、护眼、古色古香的“宣纸”沉浸阅读模式。摒弃一切广告、侧边栏和视觉干扰，还你最纯粹的阅读心流。

## ✨ 核心特性

* **🎨 古风宣纸视觉体验**：采用低对比度旧麻布背景（`#E5DEC9`）与宣纸主轴（`#F8F4E6`），搭配朱砂红点缀，极大缓解长时间阅读的视觉疲劳。
* **⚡ 毫秒级无缝热插拔**：支持随时进入/退出沉浸模式，**无需刷新页面**，完美保留当前阅读进度和滚动条位置。
* **🛡️ 完美兼容知乎原生功能**：保留知乎的“操作栏”（点赞、收藏）与“目录”按钮。目录按钮化身右上角古风方印，不仅美观，且完整保留原生点击交互逻辑。
* **⌨️ 极客快捷键**：全局监听 `Ctrl + E` (或 `Cmd + E`)，一键“展卷”与“收卷”。
* **🧹 智能垃圾清理**：自动腰斩文章底部的引流推荐、文中植入广告与购物卡片。

## 🚀 使用说明

### 方法一：控制台临时体验

1. 在知乎文章或回答页面，按 `F12` 打开开发者工具。
2. 切换到 `Console` (控制台) 面板。
3. 粘贴本脚本代码并回车，页面将瞬间“展卷”。

### 方法二：油猴脚本（推荐长期使用）

1. 安装浏览器插件 `Tampermonkey`。
2. 新建脚本，将代码粘贴进去。
3. 设置匹配网址为 `*://*.zhihu.com/*`，保存即可。

## 🕹️ 交互指南

* **展卷/收卷**：键盘按下 `Ctrl + E`。
* **退出沉浸**：点击屏幕左下角的【退出沉浸】按钮。
* **查看目录**：点击屏幕右上角的【红色方印】图标（仅当原文章包含目录时出现）。
